tailwind安裝

安裝 Tailwind CSS 和它的平行依賴套件 PostCSS 及 Auto Prefixer。

      
        npm install -D tailwindcss postcss autoprefixer

產生 tailwind.config.js和postcss.config.js 設定檔

-p 產生 postcss.config.js 設定檔

      
        npx tailwindcss init -p

tailwind.config

content:設定要編譯的檔案範圍

theme:客製化的設定

      
        //postcss.config.js 
        module.exports = {
          content: [
          "./index.html",
          "./src/**/*.{js,ts,jsx,tsx,vue,css}",
          ],
          theme: {
            extend: {},
          },
          plugins: [],
        }

postcss.config：會放 postcss 插件的設定。

開一個新的檔案，命名為 index.css。放入以下語法來載入 Tailwind CSS

      
        @tailwind base;
        @tailwind components;
        @tailwind utilities;

main.js 引入 Tailwind CSS

      
        import './index.css'

App.vue 使用

      
        <h1 className="text-3xl font-bold underline bg-sky-700 bg-green-800">
          Hello world!
        </h1>
