tailwind安裝

建立 package.json

-y 表示跳過所有提示步驟

      
        npm init -y

安裝 Tailwind CSS 和它的平行依賴套件 PostCSS 及 Auto Prefixer。

      
        npm install -D tailwindcss postcss autoprefixer

產生 tailwind.config.js 設定檔

-p 參數是告訴 Tailwind CLI 工具，順便產生 postcss.config.js 設定檔。

會放相關的 html 檔案與 js 檔案路徑，或者一些客製化的設定。

      
        npx tailwindcss init -p

新增postcss.config.js

會放 postcss 插件的設定。

引用Tailwind CSS

開一個新的檔案，命名為 index.scss。放入以下語法來載入 Tailwind CSS

      
        @tailwind base;
        @tailwind components;
        @tailwind utilities;

新增一個 index.html

      
        <head>
          ...
          <link href="style.css" rel="stylesheet">
        </head>
        <body>
          <h1 class="text-3xl font-bold underline">
            Hello world!
          </h1>
        </body>

掃描編譯的檔案範圍

修改 tailwind.config.js

content 就是設定要編譯的檔案範圍

      
        //postcss.config.js 
        module.exports = {
          content: [
            "./*.php",
            "./classes/*.php",
            "./inc/*.php",
            "./template-parts/**/*.php",
          ],
          theme: {
            extend: {},
          },
          plugins: [],
        }

編譯打包檔案

設定打包指令到 package.json 中，以下這一行指令加到 script 段落。

      
        'scripts':{
          //進行編譯
          "build": "tailwind build ./assets/sass/index.scss -o ./style.css"
          //存擋後自動編譯
          "watch": "tailwindcss -i ./src/tailwind.css -o ./dist/style.css --watch"
        }

執行指令

      
        npm run build
        npm run watch

TailwindCSS在vite项目中使用

vite.config.ts配置

      import tailwindcss from  tailwindcss
      import autoprefixer from 'autoprefixer
      
      css: {
        postcss: {
          plugins: [
            tailwindcss, 
            autoprefixer,
          ]
        }
      }

引入Tailwind的基本指令

新增 tailwind.css

      @tailwind base;
      @tailwind components;
      @tailwind utilities;

main.js

      import './tailwind.css'

引用

      <h1 class="text-3xl font-bold underline">
        Hello world!
      </h1>
