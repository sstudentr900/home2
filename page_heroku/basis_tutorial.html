建立資料夾 mkdir

      
        mkdir fileName
      cd fileName

git

git 版本

      
        git --version

移除提交的資料

      
        git rm -rf --cached .

显示所有远程仓库

      
        git remote -v

显示某个远程仓库的信息

      
        git remote show

删除和修改远程仓库

      
        git remote rm name # 删除远程仓库
      git remote rename old_name new_name # 修改仓库名

查看分支

  • refs/heads/master => 推送到 master
  • refs/heads/main => 推送到 main
      
        git show-ref

強制push

      
        //參數 -f 等同於 --force，表示強制的意思
      git push heroku main --force

看目前免費的時數

      
        heroku ps -a nodelinebotstock

heroku 版本

      
        heroku -v

heroku 登入

      
        heroku login

新增heroku

      
        //自動命名
      heroku create

      //自訂名稱
      heroku create yourAppName

      //自訂名稱
      heroku apps:create yourAppName

      //修改專案名稱
      heroku apps:rename --app [old-name] [new-name]
