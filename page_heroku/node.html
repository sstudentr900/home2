1.手動建立應用程式

右上角「 New 」中的「 Create new app 」建立第一個應用程式

2.建議專案

    
        npm init -y

3.新增忽略

.gitignore 檔案並新增忽略規則

4.新增 .Procfile 檔

heroku用來定義檔案從web底下start=>(node index.js)開啟

    
      //.Procfile 檔
      web: npm start

5.設定 package.json 檔

    
      {
        "scripts": {
          //給Procfile檔呼叫
          "start": "node index.js"
        },
      }

6.變數設定 process.env

7.push_heroku

解決跨域問題

    
      //index.js
      const cors = require("cors");
      // 處理跨域
      app.use(cors());
