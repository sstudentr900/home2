scoped

scoped會讓該元件在編譯時被添加一個以「data-v-」開頭代表唯一值

對於子元件根部的 CSS，父元件 CSS 會優先於子元件的 CSS

Message template 不是用DIV包會被父元件和它自己的 scoped 影響

deep selector

無視 scoped 的限制 子物件會吃到父物件CSS

      
        .home:deep .message {
          color: red;
        }

把前面的父元件省略

      
        :deep .message {
          color: red;
        }
