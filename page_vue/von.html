「@」v-on：監聽事件

事件修飾符

  1. capture 以捕獲形式觸發事件
  2. stop => stopPropagation 停止觸發上層 DOM 元素事件。
  3. prevent => preventDefault 阻擋預設行為
  4. self => 只有觸發此 DOM 元素本身才會觸發 self 事件。
  5. once => 此事件只觸發一次。
  6. passive => 無視 prevent 功能。
  7. capture => 會優先觸發。

指定事件只觸發一次(@click.once)

prevent 阻擋預設行為 event.preventDefault()

@input事件是实时监控的，每次输入都会调用

  1. @keyup.enter需要点击回车键触发
  2. @change
  3. @blur

一般在元件內的input使用v-model

@input監聽表單元素的即時變化

@change監聽值改變
