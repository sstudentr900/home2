position

  1. static 靜態定位
  2. relative 相對定位:相對於標準流中位置移動
  3. absolute 絕對定位:跟父級位置移動,不佔位
  4. fixed 固定定位:跟窗口做偏移

文字在盒中位置

            
                text { 
                    height: 30px;
                    line-height: 30px; //垂直置中
                    text-align: center:
                }

textarea 不能拖移

            
                textarea { 
                    resize: none;
                }
