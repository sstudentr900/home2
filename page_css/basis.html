position

  1. static 靜態定位
  2. relative 相對定位:相對於標準流中位置移動
  3. absolute 絕對定位:跟父級位置移動,不佔位
  4. fixed 固定定位:跟窗口做偏移

文字在盒中位置

    
      <style>
        text {
          height: 30px;
          line-height: 30px;
          text-align: center;
        }
      </style>

textarea 不能拖移

    
      <style>
        textarea {
          resize: none;
        }
      </style>

匯入

    
      <style>
        /*匯入文字*/
        @import url(https://fonts.googleapis.com/css?family=Lato:400,700,300);
