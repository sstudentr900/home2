Grid

  1. Flex是一維的，排版方式是「由上到下」 而grid是二維的網格系統，可以「指定」內元件排版在網格內的「任何地方」
  2. fr代表剩餘空間「欲分配的比例」，Ex：有一剩餘空間為100px，0.7fr和0.3fr代表將100px分成兩行，其行空間分別為70px和30px
  3. repeat()重複模式的列或行，Ex：repeat(2,60px)=>60px 60px

grid-template(設定container內欲分配的網格空間)

grid-template 縮寫是 grid-template-rows / grid-template-columns

    
      <style>
        .grid_container {
          grid-template-columns: 30px 100px 0.7fr 0.3fr;
          grid-template-rows: 100px 150px 0.2fr 0.8fr;
        }

        .grid_container {
          grid-template: 100px 150px 0.2fr 0.8fr / 30px 100px 0.7fr 0.3fr;
        }


        .grid_container {
          grid-template-columns:60px 60px 60px 60px; /* 也可寫成repeat(4, 60px) */
          grid-template-rows:35px 35px; /* 也可寫成repeat(2, 35px) */
        }

        .grid_container {
          grid-template-columns:repeat(4, 60px);
          grid-template-rows: repeat(2, 35px);
        }

        .grid_container {
          grid-template:repeat(4, 60px) / repeat(2, 35px);
        }
      </style>

grid-column 和 grid-row (設定內元件的位置)

  1. grid-column(是x起始,結束位置) 縮寫是 grid-column-start / grid-column-end
  2. grid-row(是y起始,結束位置) 縮寫是 grid-row-start / grid-row-end
    
      <style>
        .box_red {
          grid-column-start: 3;
          grid-column-end: 5; /*x結束位置(1是1列,2是1列,3是2列)*/
          grid-row-start: 1;
          grid-row-end: 2;
        }
        .box_red {
          grid-column:3/5;
          grid-row:1/2; 
        }
      </style>

grid-gap X,Y間距

縮寫是 grid-column-gap grid-row-gap

    
      <style>
        .grid_container {
            grid-column-gap:30px;
            grid-row-gap:15px;
        }
        .grid_container {
            grid-gap:30px 15px;
        }
      </style>

grid-area

縮寫是 grid-row-start / grid-column-start / grid-row-end / grid-column-end

    
      <style>
        .box_red {
          grid-column:3/5;
          grid-row:1/2;
        }
        .box_red {
          grid-area:1/3/2/5;
        }
      </style>

justify 水平方向

  1. justify-items: stretch | start | end | center
  2. Justify-self: 和justify-items的功能一樣，但它是針對「單一」對象

align 垂直方向

  1. align-items:stretch | start | end | center | baseline;
  2. align-self: 和align-item的功能一樣，但它是針對「單一」對象

place

  1. place-items 縮寫是 align-items justify-item
  2. place-self 縮寫是 align-self justify-self
  3. place-content 縮寫是 align-content justify-content
