height:100vh全屏佈局,大多數移動設備上的 Chrome 和 Firefox 瀏覽器在頂部都有一個 UI（地址欄等）。在 Safari 上，更棘手的地址欄位於底部。不同的瀏覽器有不同大小的視口移動設備計算瀏覽器視口為（頂部欄 + 文檔 + 底部欄）= 100vh 使用 100vh 將整個文檔填充到頁面

解決方案 min-height: 100vh;
