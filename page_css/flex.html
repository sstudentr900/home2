flex-wrap:

  1. nowrap (預設值)
  2. wrap (換行)
  3. wrap-reverse (換行，且行順序反轉)

flex-flow

  1. flex-flow是flex-direction和flex-wrap的縮寫
  2. flex-flow:column wrap;

justify-content

  1. Flex-start
  2. Flex-end
  3. Center 中間
  4. Space-between 是平均分配「元素」位置，但左右兩側會貼齊主軸的起點、終點
  5. Space-around 是平均分配「元素」位置，但左右兩側的間距會較小
  6. Space-evenly 是「元素」和「左右兩側」的間距皆相同

align-item

  1. Stretch Stretch是align-item的預設值，會往交錯軸方向將內元素全部填滿 如果你有設置height，則不會起作用
  2. flex-start 從交錯軸的「起點」開始排
  3. flex-end 從交錯軸的「終點」開始排
  4. center
  5. baseline

flex-direction

  1. row (預設值，由左至右，從上到下)
  2. row-reverse (排列方向和row一樣，但內元件順序會反轉)
  3. column (先從上到下，再由左至右)
  4. column-reverse (排列方向和column一樣，但內元件順序會反轉)

Flex (flex-grow、flex-shrink、flex-basis)

  1. Flex-grow 預設值為0，它會依照你設定的值去分配剩餘空間
  2. Flex-shrink 壓縮比例，預設值為1，代表大家所壓縮的比例一樣

Responsive Gap for Flexbox

place-content 需搭配 grid

      
        <style>
            .flex {
              gap: 10px; /*X,Y間距*/
              display: flex;
            }
            .col-4 {
              flex: calc((100% - 30px)/4); /* 30px => (10px間距)*(4-1) */
              /* flex: 1 1 calc((100% - 30px)/4); 同上 */
              height: 100px;
              display: grid;
              place-content: center; /*center center*/
              background-color: #fff;
              padding: 10px 15px;
            }
        </style>
