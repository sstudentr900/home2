輔助插件

  1. Open in Browser====>右键在浏览器打开
  2. indent-rainbow=====>代碼間距提示

快捷键

  1. Ctrl + K + 1===========>摺疊第一層(2是二層，3是三層)
  2. Ctrl + G===============>跳转行
  3. Ctrl + D===============>匹配当前选中的词汇或者行，再次选中-可操作
  4. Ctrl + K Ctrl + D======>移动当前选择到下个匹配选择的位置(光标选定)
  5. ctrl+K ctrl+S==========>快捷键全部
  6. alt+z==================>換行
  7. Shift + Alt + F========>自動格式
