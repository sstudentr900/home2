yum安裝

yum安裝

        
            yum install 套件名稱

yum安裝所有相關套件(加*)

        
            yum install 套件名稱*

清單

        
            yum list

移除相關套件

        
            yun remove 套件名稱

清除暫存資料夾下安裝

        
            yum clean 套件名稱

搜尋套件

        
            yum search 套件名稱

套件資訊

        
            yum info 套件名稱
