# GitHub 上傳
## 1.註冊檔案或目錄到索引 '.'目錄裡的所有檔案註冊到索引裡
```
    git add .
```

## 2.提交資訊
```
    git commit -m 'customText'
```

## 3.push到GitHub上面 origin 表示遠端，master 表示分支名(第一次)
```
    git push -u  origin master
```

## 4.push到GitHub上面
```
    git push
```
