# Command Line 常用指令整理
## 位址相關
```
    //顯示目前所在位置
    pwd 

    //顯示目前所在位置的所有資料
    ls

    //顯示目前所在位置的所有資料即隱藏資料
    ls –al

    //跳到其他資料夾 例：cd \Users\Asus\Desktop （跳到桌面）
    cd

    //跳出上一層
    cd ..

    //清除
    clear          
```
## 檔案相關
```
    //新增檔案並輸入文字
    echo "text" > fileName

    //新增檔案
    cd > fileName

    //刪除檔案
    rm fileName

    //複製檔案
    cp cloneFile  pasteFile

    //新增資料夾
    mkdir fileName

    //刪除資料夾
    rm fileName

    //複製資料夾
    cp –r cloneFile pasteFile
```
