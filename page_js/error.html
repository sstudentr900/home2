Error

  1. RangeError(超過範圍)
  2. ReferenceError(找不到變數：拼錯字)
  3. SyntaxError(語法錯誤)
  4. TypeError（找不到函式）
  5. URIError 當使用 encodeURI() 或 decodeURI() 的方法，但確有給了不合法的 URI 時會拋出這個錯誤
  6. EvalError

Error 客製化例外情況

      
        <script>
          var error = new Error('error message');
          console.log(error); // Error: error message
        </script>

throw

      
        <script>
          throw 'Error2'; // String type
          throw 42; // Number type
          throw true; // Boolean type
        </script>

try 和 catch

      
        <script>
          try {
              //在这里运行代码
              throw 'error'
          } catch (e) {
              //处理错误
              console.log('錯誤輸出')
          } finally {
              //完成執行
              console.log('任何情況都錯誤輸出')
          }
        </script>

foreach try catch 挑出迴圈

      
        <script>
          try {
            var array = [1, 2, 3, 4, 5, 6];
            //第3次，结束迴圈
            array.forEach((value) => {
              console.log("value---->", value);
              if (value > 3) {
                throw new Error("抛出异常跳出")
              }
            });
          } catch (e) {
            console.log(e)
          };
        </script>
