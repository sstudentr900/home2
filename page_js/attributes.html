attribute

hasAttribute 方法用來檢查 HTML 元素是否有某個屬性。

    
        document.getElementById('a').hasAttribute('href')

getAttribute 方法用來取得 HTML 元素的屬性值

    
        document.getElementById('a').getAttribute('href')

setAttribute 方法用來新增 HTML 元素的屬性

    
        document.getElementById('a').setAttribute('target', '_blank')

removeAttribute 方法用來移除 HTML 元素的某個屬性。

    
        document.getElementById('a').removeAttribute('data-foo')

attributes 取得 HTML 元素上所有的屬性

    
        document.getElementById('a').attributes['href'].value
        document.getElementById('a').attributes['id'].value
