Access-Control-Allow-Origin

伺服器必須在回應表頭(response header)裏加上Access-Control-Allow-Origin，並在Access-Control-Allow-Origin的設定裏，新增我的Origin(即是我的網址)，或者設定為萬用字符*，代表所有Origin都接受，這是在公共API裏常見的設定。

        // Access-Control-Allow-Origin: https://amazing.site
        //或者
        // Access-Control-Allow-Origin: *
