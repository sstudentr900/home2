輔助插件

  • GROUP_CONCAT(要合併的值)
  • FIND_IN_SET(查詢的值,查詢的表)

SELECT m.id,(SELECT GROUP_CONCAT(c.name) FROM c WHERE FIND_IN_SET(c.id,m.c_id)) AS className FROM m;

m 主 ----------------- id c_id ----------------- 1 1,2 2 2,3 c 類 ----------------- id name ----------------- 1 A 2 B 3 C 結果 ----------------- id className ----------------- 1 A,B 2 B,C
