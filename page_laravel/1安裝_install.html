安裝錯誤 PHP error: "The zip extension and unzip command are both missing, skipping."

對於 Windows，如果您使用的是 XAMPP 等語言環境開發環境，只需找到 php.ini 檔案（在我的例子中為 C:\xampp\php），打開它並刪除以下行中的分號。

      
        ;extension=zip

初识 Laravel 11

目录结构

      
        |---- app/  应用目录
        |---- bootstrap/ 启动目录
        |---- config/ 配置目录
        |---- database/ 数据迁移目录
        |---- public/ 入口
        |---- resources/ 资源目录，主要是视图，静态文件
        |---- routes/ 路由
        |---- storage/ 存储目录
        |---- tests/ 测试
        |---- vendor/ 依赖目录
        |---- vite.config.js vite 配置
        |---- artisan
        |---- composer.json PHP 依赖
        |---- package.json 前端依赖
        |---- phpunit.xml
        |---- README.md

app 目录

      
        ├─app 
        │  ├─Http
        │  │  └─Controllers
        │  │          Controller.php
        │  │         
        │  ├─Models 
        │  │      User.php 
        │  └─Providers 
        │          AppServiceProvider.php
