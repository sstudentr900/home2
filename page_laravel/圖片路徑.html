引用 Vite 編譯的圖片

      
        vite.config.js
        import { defineConfig } from 'vite';
        import laravel from 'laravel-vite-plugin';

        export default defineConfig({
            plugins: [
                laravel({
                    input: [
                        'resources/css/app.css',
                        'resources/js/app.js',
                        'resources/images/logo.png', // 假設圖片需要透過 Vite 處理
                    ],
                    refresh: true,
                }),
            ],
        });

        //bland
        <img src="{{ Vite::asset('resources/images/logo.png') }}">

引用 public 目錄中的靜態圖片（未經 Vite 處理）

      
        //public/images/static-logo.png

        <img src="{{ asset('images/static-logo.png') }}">

url() 函數

用途：用於生成應用程式中的動態 URL，例如路由、控制器動作或其他非靜態資源的路徑。

      
        <a href="{{ url('home') }}">Home</a>

圖片維持顯示比例

圖片一定要加aspect-ratio 這樣高度才會再載入時撐高

      
        <img class="aspect-[359/320]" src="{{url('/images/..')}}" loading="lazy">
