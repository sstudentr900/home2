Laravel 11 目录结构

      
        |---- app/  应用目录
        |---- bootstrap/ 启动目录
        |---- config/ 配置目录
        |---- database/ 数据迁移目录
        |---- public/ 入口
        |---- resources/ 资源目录，主要是视图，静态文件
        |---- routes/ 路由
        |---- storage/ 存储目录
        |---- tests/ 测试
        |---- vendor/ 依赖目录
        |---- vite.config.js vite 配置
        |---- artisan
        |---- composer.json PHP 依赖
        |---- package.json 前端依赖
        |---- phpunit.xml
        |---- README.md

app 目录

      
        ├─app 
        │  ├─Http

        │  │          Controller.php
        │  │         
        │  ├─Models

專案架構介紹

      
        - app/
      - Console/
      - Commands/ #Laravel的命令檔案
      - Kernel.php #Laravel的排程
      - Http/
      - Controller/ #控制器
      - Middleware/ #中介層
      - bootstrap/ #放有laravel的快取
      - config/ #各種參數設定
      - database/ #資料庫相關的東西，例如migrations
      - migrations/ #資料庫相關migration檔案
      - node_modules/ #專案套件的安裝目錄
      - public/ #對外公開的目錄
      - resources/ #放置前端的檔案，view
      - js/ #Js原始檔案所在目錄
      - lang/ #語系檔案所在目錄
      - sass/ #SASS/SCSS檔案所在目錄
      - views/ #前端模板所在目錄
      - route/ #路由器，設定路由的地方
      - storage/ #放置log、上傳的檔案
      - tests/ #放置測試檔案
      - vendor/ #Composer套件安裝目錄
      - .env #環境變數設定檔案
      - composer.json #Composer套件安裝清單
      - package.json #專案套件安裝清單
