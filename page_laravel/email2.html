安裝郵件插件

    
      composer require guzzlehttp/guzzle

郵件相關設定

郵件的設定放在config/mail.php檔案中

    
      'smtp' => [
          'transport' => 'smtp',
          'host' => 'smtp.gmail.com',
          'port' => 587,
          'encryption' => 'tls',
          'username' => 'gmail得信箱',
          'password' => 'gmail的應用程式密碼',
          'timeout' => null,
          'auth_mode' => null,
      ],
      'from' => [
          'address' => '寄件人信箱',
          'name' => '寄件人名稱',
      ],
