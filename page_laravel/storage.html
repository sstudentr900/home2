Laravel安裝

cmd執行下列命令來安裝Laravel

            
                composer global require "laravel/installer"

建立專案

  1. LaravelBlog 檔案名稱
  2. create-project 創建專案
  3. --prefer-dist 當有可用的包時，從 dist 安裝
            
                composer create-project --prefer-dist laravel/laravel LaravelBlog

啟動專案serve

            
                cd LaravelBlog
                php artisan serve

npm 安裝

            
                npm install

composer 安裝

            
                composer install

即時更新請看mix

專案架構介紹

            
                - app/
                - Console/
                    - Commands/   #Laravel的命令檔案
                    - Kernel.php  #Laravel的排程
                - Http/
                    - Controller/ #控制器
                    - Middleware/ #中介層
                - bootstrap/      #放有laravel的快取
                - config/         #各種參數設定
                - database/       #資料庫相關的東西，例如migrations
                    - migrations/ #資料庫相關migration檔案
                - node_modules/   #專案套件的安裝目錄
                - public/         #對外公開的目錄
                - resources/      #放置前端的檔案，view
                    - js/         #Js原始檔案所在目錄
                    - lang/       #語系檔案所在目錄
                    - sass/       #SASS/SCSS檔案所在目錄
                    - views/      #前端模板所在目錄
                - route/          #路由器，設定路由的地方   
                - storage/        #放置log、上傳的檔案 
                - tests/          #放置測試檔案
                - vendor/         #Composer套件安裝目錄
                - .env            #環境變數設定檔案
                - composer.json #Composer套件安裝清單
                - package.json #專案套件安裝清單
