env 變數引用

引用 .env 檔案中的 MAIL_FROM_ADDRESS 變數

使用 config() 輔助函數

Laravel 會自動從 .env 檔案載入 MAIL_FROM_ADDRESS 到 config/mail.php 中（通常在 'from' => ['address' => env('MAIL_FROM_ADDRESS', null), ...]）

      
        //Controller 中這樣引用：
        $mailFromAddress = config('mail.from.address');

自訂.env變數

在 config 資料夾中建立一個新文件custom.php

要在執行 php artisan config:cache 清緩存

      
        //.env：
        CUSTOM_VARIABLE=my_value

        //在 config/custom.php 中添加以下內容：
        return [
          'custom_variable' => env('CUSTOM_VARIABLE', 'default_value'),
        ];

        //在 Controller 中引用：
        $customValue = config('custom.custom_variable');
