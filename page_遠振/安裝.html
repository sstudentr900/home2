找不到路徑有可能是被cache

刪除 bootstrap/cache 中的檔案（如 config.php 或 services.php）

bootstrap/cache/config.php

儲存所有配置檔案（config/*.php）的編譯版本。Laravel 會將 config 目錄下的配置合併成單一檔案，加快配置載入速度。如果你在 Windows 上生成快取，config.php 可能包含反斜線 \，導致 Linux 環境無法正確解析路徑。

bootstrap/cache/services.php

儲存 Composer 自動載入的服務提供者清單。包含專案中使用的服務提供者和相關類的快取，加速自動載入。

bootstrap/cache/packages.php（某些情況下）

儲存 Laravel 套件的快取資訊。如果你使用了第三方套件，可能會生成此檔案。
